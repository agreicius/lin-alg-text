Item 2.

Suppose \(\boldw_1,\boldw_2\in \range(T)\text{.}\) This means there are \(\boldv_1, \boldv_2\in V\) such that \(T(\boldv_i)=\boldw_i\) for \(i=1,2\text{.}\) We must show \(\boldw=\boldw_1+\boldw_2\in\range(T)\text{.}\) Set \(\boldv=\boldv_1+\boldv_2\text{.}\) Then

\begin{align*} T(\boldv)\amp =\amp T(\boldv_1+\boldv_2)\\ \amp =\amp T(\boldv_1)+T(\boldv_2) \ \text{ (since \(T\) is a linear transformation) }\\ \amp =\amp \boldw_1+\boldw_2=\boldw\text{.} \end{align*}

Since we have provided a \(\boldv\) with \(T(\boldv)=\boldw_1+\boldw_2\text{,}\) we see that \(\boldw_1+\boldw_2\in\range(T)\text{.}\)
