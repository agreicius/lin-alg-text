Matrices.
Definition 1.4.1.

A matrix is a rectangular array of numbers.

\begin{equation*} A=\genmatrix \end{equation*}

The size (or dimension) of a matrix is given by the pair \(m\times n\text{,}\) where \(m\) is the number of rows, and \(n\) is the number of columns. The matrix \(A\) above is an \(m\times n\) matrix.
