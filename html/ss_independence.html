Section 3.4 Linear independence

Chapter 3. Section 3.4: executive summary.
Definitions:.

linear independence, the Wronskian \(W(f_1,f_2,\dots,f_n)(x)\text{.}\) \alert{Procedures:} how to determine whether a set of vectors is independent. \alert{Theorems:} if \(f_1,f_2,\dots, f_n\) are linearly dependent, then \(W(f_1,f_2,\dots,f_n)(x)=0\) for all \(x\text{.}\)

Chapter 3. Section 3.4: linear independence.
Definition 3.4.1.

Let \(V\) be a vector space. A set \(S=\{\boldv_1,\dots ,\boldv_r\}\) of elements of \(V\) is linear independent if

\begin{equation*} k_1\boldv_1+k_2\boldv_2+\cdots k_r\boldv_r=\boldzero \Rightarrow k_i=0 \text{ for all \(i\) }\text{.} \end{equation*}

In other words, the only linear combination of the \(\boldv_i\) yielding \(\boldzero\) is the trivial linear combination we get by setting \(k_i=0\) for all \(i\text{.}\)

The set \(S\) is linearly dependent if it is not linearly independent; i.e., if we can find a nontrivial linear combination of the \(\boldv_i\) that yields \(\boldzero\text{.}\)

Theorem 3.4.2.

The set \(S\) is linearly independent if and only if no element \(\boldv_i\) can be written as a linear combination of the other \(\boldv_j\text{.}\) Similarly, \(S\) is linearly dependent if and only if one of the elements \(\boldv_i\) can be written as a linear combination of the remaining \(\boldv_j\text{.}\)

Example in $P_n$.
General procedure:.

all questions of linear dependence can be boiled down to deciding whether a certain linear system can be solved or not! Let \(S=\{x^2+x-2, 2x^2+1, x^2-x\}\subset P_2\text{.}\) Decide whether \(S\) is linearly independent.

Proof.

First observe that \(\boldzero=0x^2+0x+0\text{,}\) the zero polynomial.

We ask whether there is a nontrivial combination

\begin{align*} a(x^2+x-2)+b(2x^2+1)+c(x^2-x)\amp =\amp 0x^2+0x+0\\ (a+2b+c)x^2+(a-c)x+(-2a+b)\amp =\amp 0x^2+0x+0 \end{align*}

Example in $M_{mn}$.
General procedure:.

all questions of linear dependence can be boiled down to deciding whether a certain linear system can be solved or not! Let \(S=\left\{ A_1=\begin{bmatrix}3\amp 1\\ 2\amp -3 \end{bmatrix} , A_2= \begin{bmatrix}0\amp 4\\ 2\amp 0 \end{bmatrix} , A_3=\begin{bmatrix}-2\amp -2\\ -2\amp 2 \end{bmatrix} \right\}\subset M_{22}\text{.}\) Decide whether \(S\) is linearly independent.

Example in function space.
General procedure:.

all questions of linear dependence can be boiled down to deciding whether a certain linear system can be solved or not! Let \(S=\{f(x)=x, g(x)=\cos(x), h(x)=\sin(x)\}\subseteq C^\infty(\R)\text{.}\)

Decide whether \(S\) is linearly independent.

Proof.

First observe that \(\boldzero\) is the zero function: the function that assigns 0 to all inputs \(x\text{.}\)

We ask whether there is a nontrivial combination \(af+bg+ch=\boldzero\text{.}\)

Key observation: the equality above is an equality of functions. Thus this is true if and only if \(af(x)+bg(x)+ch(x)=0\) \alert{for all \(x\)}.

To get a linear system, we evaluate the above at a few clever choices of \(x\text{:}\)

\begin{align*} x=0\amp :\amp a(0)+b\cos(0)+c\sin(0)=0\Rightarrow 0+b+0=0\Rightarrow b=0\\ x=\pi\amp :\amp a(\pi)+c\sin(\pi)=0\Rightarrow \pi a+0=0\Rightarrow a=0 \end{align*}

Having shown \(a=b=0\text{,}\) we are left with the equation \(c\sin(x)=0\) for all \(x\text{,}\) which is true iff \(c=0\text{.}\)

Thus the only linear combination yielding \(\boldzero\) is \(a=b=c=0\text{,}\) the trivial one, showing \(S\) is linearly independent.

Linear independence in function spaces.

As the last example illustrates, a set of functions \(S=\{f_1,f_2, \dots ,f_r\}\) is linearly independent if

\(k_1f_1+k_2f_2+\cdots k_rf_r=\boldzero\) implies \(k_i=0\) for all \(i\text{.}\)

Recall \(\boldzero\) here stands for the zero function. Thus we must treat such a linear combination as a function identity! In other words to say

\begin{equation*} k_1f_1+k_2f_2+\cdots k_rf_r=\boldzero \end{equation*}

is simply to say that

\begin{equation*} k_1f_1(x)+k_2f_2(x)+\cdots k_rf_r(x)=0 \end{equation*}
for all \(x\).

in the given domain.

The beauty of this “for all \(x\)” is that by picking say \(r\) actual examples of \(x\) and evaluating above, we generate \(r\) linear equations in the unknowns \(k_i\text{.}\)

If this system has no nontrivial solutions, then we know the functions are independent.

However, if this system DOES have a nontrivial solution we CANNOT conclude the functions are linearly dependent. Why? We would have shown the identity above holds only for these \(r\) choices of \(x\text{,}\) but not necessarily all \(x\text{!}\)

Theorem 3.4.4. Wronskian.

Let \(V=C^{\infty}(X)\text{,}\) where \(X\) is a fixed interval (usually \(X=\R\)), and let \(S=\{f_1,f_2,\dots ,f_n\}\) be a set of elements of \(V\text{.}\) Let \(W(x)\) be the Wronskian of the \(f_i\text{.}\) Then

\begin{equation*} W\ne\boldzero\Rightarrow \text{ \(S\) is linearly indendent }\text{.} \end{equation*}
Comments.

(1) Again \(\boldzero\) here is the zero function. So \(W\ne\boldzero\) means there is an \(x\) in \((a,b)\) such that \(W(x)\ne 0\text{.}\)

(2) This implication only goes one way!! In other words, \(W(x)=0\) for all \(x\) does not imply \(S\) is dependent!!
