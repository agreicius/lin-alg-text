Section 4.2 Linear transformations and bases

Linear transformations and bases.

As the next theorem articulates, a linear transformation \(T\colon V\rightarrow W\) is uniquely determined by how it acts on the elements of a basis for \(V\text{.}\)

Theorem 4.2.1.

Let \(B=\{\boldv_1,\dots,\boldv_n\}\) be a basis for \(V\text{,}\) and let \(W\) be any space.

  1. Given any choice of \(n\) elements \(\boldw_1,\boldw_2,\dots, \boldw_n\in W\) there is a unique linear transformation \(T\colon V\rightarrow W\) such that \(T(\boldv_i)=\boldw_i\text{.}\)

  2. Given linear transformations \(T_1,T_2\colon V\rightarrow W\text{,}\) \(T_1=T_2\) if and only if \(T_1(\boldv_i)=T_2(\boldv_i)\) for \(1\leq i\leq n\text{.}\)

This is an extremely useful theorem as it allows us, given a basis \(B\) of \(V\text{,}\) to

  1. easily define linear transformations simply by declaring where \(\boldv_i\) gets sent for each \(i\text{,}\) and

  2. easily check whether two linear transformations are equal simply by checking that they agree on the basis elements \(\boldv_i\text{.}\)

Proof.

First observe that given any \(\boldv=c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n\text{,}\) if \(T\) is a linear transformation satisfying \(T(\boldv_i)=\boldw_i\) for all \(\boldv_i\in B\text{,}\) then we must have

\begin{align*} T(\boldv)\amp =T(c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n)\\ \amp =c_1T(\boldv_1)+c_2T(\boldv_2)+\cdots +c_nT(\boldv_n)\\ \amp =c_1\boldw_1+c_2\boldw_2+\cdots +c_n\boldw_n \end{align*}

This shows that there is at most one such \(T\text{,}\) and furthermore that \(T_1=T_2\) if and only if \(T_1(\boldv_i)=T_2(\boldv_i)\) for all \(\boldv_i\in B\text{.}\)

Thus we have proven (b), and the proof of (a) is completed by showing that the formula \(T(\boldv)=c_1\boldw_1+c_2\boldw_2+\cdots +c_n\boldw_n\text{,}\) where \(\boldv=c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n\text{,}\) defines a linear transformation satisfying \(T(\boldv_i)=\boldw_i\text{.}\)

The latter property is obvious, since \(\boldv_i=1\boldv_i+\sum_{j\ne i}0\boldv_j\) implies \(T(\boldv_i)=1\boldw_i+\sum_{i\ne j}0\boldw_j=\boldw_i\text{.}\)

Now let's show \(T\) is linear: given \(\boldv=\sum_{i=1}^n c_i\boldv_i\) and \(\boldv'=\sum_{i=1}^nc_i'\boldv_i\text{,}\) we have \(r\boldv+s\boldv'=\sum_{i=1}^n(rc_i+sc_i')\boldv_i\text{,}\) in which case

\begin{align*} T(r\boldv+s\boldv')\amp =\sum_{i=1}^n(rc_i+sc_i')\boldw_i \amp \text{ (def. of \(T\)) }\\ \amp =r\sum_{i=1}^n c_i\boldw_i+s\sum_{i=1}^nc_i'\boldw_i=rT(\boldv)+sT(\boldv')\ \checkmark \amp \text{ (def. of \(T\)) } \end{align*}
Transformations of $\R^n$.
Corollary 4.2.2.

Let \(T\colon \R^n\rightarrow \R^m\) be a linear transformation. Then \(T=T_A\) for some \(\underset{m\times n}{A}\text{.}\)

In other words, any linear transformation whose domain is \(\R^n\) for some \(n\) and whose codomain is \(\R^m\) for some \(m\text{,}\) is in fact a matrix transformation \(T=T_A\text{.}\)

Proof.

Our proof includes a recipe for computing the matrix \(A\) such that \(T=T_A\text{.}\)

Namely, given a linear \(T\colon\R^n\rightarrow \R^m\text{,}\) let \(A\) be the matrix whose \(j\)-th column is \(\boldc_j=T(\bolde_j)\text{,}\) where \(\bolde_j\) is the \(j\)-th element of the standard basis of \(\R^n\text{.}\)

I claim \(T=T_A\text{.}\)

According to the Theorem 4.1.1, I only need to show that \(T\) and \(T_A\) agree on a basis for \(\R^n\text{.}\) Let's take the standard basis \(B=\{\bolde_1,\dots,\bolde_n\}\text{.}\)

Then

\begin{align*} T_A(\bolde_j)\amp =A\bolde_j \amp \text{ (def. of \(T_A\)) }\\ \amp =\boldc_j \amp \text{ (column expansion) }\\ \amp =T(\bolde_j) \amp \text{ (def. of \(A\)) } \end{align*}

We've shown that \(T_A(\bolde_j)=T(\bolde_j)\) for all \(\bolde_j\in B\text{.}\) Theorem 4.1.1 implies \(T=T_A\text{.}\)

Linear transformations of $\R^n$.

Once we know a given function \(T\colon \R^n\rightarrow\R^m\) is linear, the recipe provided in the previous proof allows us to quickly compute the matrix \(\underset{m\times n}{A}\) such that \(T=T_A\text{:}\) simply set the \(j\)-th column of \(A\) equal to \(T(\bolde_j)\text{.}\)

Example 4.2.3.

Consider again rotation by \(\theta\text{,}\) \(T_\theta\colon \R^2\rightarrow \R^2\text{,}\) where \(\theta\) is a fixed angle.

Once we know \(T_\theta\) is linear (we do now, but this wasn't obvious), then we easily come up with the matrix \(A_\theta\) that defines \(T_\theta\text{.}\)

The first column of \(A_\theta\) is

\begin{equation*} T_\theta(\bolde_1)=T_\theta (\cos(0),\sin(0))=(\cos(\theta),\sin(\theta))\text{.} \end{equation*}

The second column of \(A_\theta\) is

\begin{equation*} T_\theta(\bolde_2)=T_\theta (\cos(\pi/2),\sin(\pi/2))=(\cos(\pi/2+\theta),\sin(\pi/2+\theta))=(-\sin(\theta), \cos(\theta))\text{.} \end{equation*}

Thus \(A_\theta=\begin{bmatrix}\cos\theta\amp -\sin\theta \\ \sin\theta\amp \cos\theta \end{bmatrix}\text{.}\)
