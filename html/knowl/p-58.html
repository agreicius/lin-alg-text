Recall that the set of solutions \((x,y)\) to a single linear equation in 2 variables constitutes a line \(\ell\) in \(\R^2\text{.}\) We denote this \(\ell\colon ax+by=c\text{.}\) Similarly, the set of solutions \((x,y,z)\) to a single linear equation in 3 variables \(ax+by+cz=d\) constitutes a plane \(\mathcal{P}\) in \(\R^3\text{.}\) We denote this \(\mathcal{P}\colon ax+by+cz=d\text{.}\)
