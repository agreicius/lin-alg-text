Exercise 1. Geometry of linear systems.

Recall that the set of solutions \((x,y)\) to a single linear equation in 2 variables constitutes a line \(\ell\) in \(\R^2\text{.}\) We denote this \(\ell\colon ax+by=c\text{.}\) Similarly, the set of solutions \((x,y,z)\) to a single linear equation in 3 variables \(ax+by+cz=d\) constitutes a plane \(\mathcal{P}\) in \(\R^3\text{.}\) We denote this \(\mathcal{P}\colon ax+by+cz=d\text{.}\)

  1. Fix \(m>1\) and consider a system of \(m\) linear equations in the 2 unknowns \(x\) and \(y\text{.}\) What do solutions \((x,y)\) to this system of linear equations correspond to geometrically?

  2. Use your interpretation to give a geometric argument that a system of \(m\) equations in 2 unknowns will have either (i) 0 solutions, (ii) 1 solution, or (iii) infinitely many solutions.

  3. Use your geometric interpretation to help produce explicit examples of systems in 2 variables satisfying these three different cases (i)-(iii).

  4. Now repeat (a)-(b) for systems of linear equations in 3 variables \(x,y, z\text{.}\)
