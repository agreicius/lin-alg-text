Linear transformations and bases.

As the next theorem articulates, a linear transformation \(T\colon V\rightarrow W\) is uniquely determined by how it acts on the elements of a basis for \(V\text{.}\)

Theorem 4.2.1.

Let \(B=\{\boldv_1,\dots,\boldv_n\}\) be a basis for \(V\text{,}\) and let \(W\) be any space.

  1. Given any choice of \(n\) elements \(\boldw_1,\boldw_2,\dots, \boldw_n\in W\) there is a unique linear transformation \(T\colon V\rightarrow W\) such that \(T(\boldv_i)=\boldw_i\text{.}\)

  2. Given linear transformations \(T_1,T_2\colon V\rightarrow W\text{,}\) \(T_1=T_2\) if and only if \(T_1(\boldv_i)=T_2(\boldv_i)\) for \(1\leq i\leq n\text{.}\)

This is an extremely useful theorem as it allows us, given a basis \(B\) of \(V\text{,}\) to

  1. easily define linear transformations simply by declaring where \(\boldv_i\) gets sent for each \(i\text{,}\) and

  2. easily check whether two linear transformations are equal simply by checking that they agree on the basis elements \(\boldv_i\text{.}\)
