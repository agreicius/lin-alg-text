Example.

Let \(T=T_A\colon \R^2\rightarrow\R^3\text{,}\) where \(A=\begin{bmatrix}1\amp 1\\ 2\amp 1\\ 3\amp 5 \end{bmatrix}\text{.}\) According to the theorem, \(\range T_A\) is a subspace of \(\R^3\text{.}\) Can we identify this subspace as a familiar geometric object?

By definition \(\range T_A\) is the set

\begin{equation*} \{\boldy\in\R^3\colon \boldy=T_A(\boldx) \text{ for some \(\boldx\in \R^3\) } \}=\left\{\boldy=\begin{bmatrix}a\\ b\\ c \end{bmatrix} \colon \boldy=A\boldx \text{ for some \(\boldx\in\R^2\) } \right\}\text{.} \end{equation*}

Thus to compute \(\range T_A\) we must determine which choice of \(\boldy=(a,b,c)\) makes the system \(A\boldx=\boldy\) consistent. We answer this using our good, old friend Gaussian elimination!

\begin{equation*} \begin{bmatrix}1\amp 1\amp a\\ 2\amp 1\amp b\\ 3\amp 5\amp c \end{bmatrix} \xrightarrow{\text{ row reduction } }\begin{bmatrix}1\amp 1\amp a\\ 0\amp 1\amp 2a-b\\ 0\amp 0\amp -7a+2b+c \end{bmatrix} \end{equation*}

To be consistent we need \(-7a+2b+c=0\text{.}\) We conclude that \(\range T\) is the set of all \((a,b,c)\) satisfying \(-7a+2b+c=0\text{.}\) Geometrically this is the plane passing through \(O\) with normal vector \(\boldn=(-7,2,1)\text{.}\)
