We wish to prove a set equality. We do so by showing the \(\subseteq\) and \(\supseteq\) relations separately. (See my proof technique guide!)

\(\subseteq\).

Note first that \(p_1(1)=p_2(1)=0\text{.}\) Given an element \(q(x)\in \Span\left(\{p_1(x), p_2(x)\}\right)\text{,}\) we have \(q(x)=ap_1(x)+bp_2(x)\) for some \(a, b\in \R\text{.}\) But then \(q(1)=ap_1(1)+bp_2(1)=0+0=0\text{.}\) Thus \(q(x)\in \{p(x)\colon p(1)=0\}\text{.}\)

\(\supseteq\).

Now take \(p(x)=a_2x^2+a_1x+a_0\in \{p(x)\in P_2\colon p(1)=0\}\text{.}\) We must find \(a, b\in \R\) such that \(p(x)=ap_1+bp_2\text{.}\)

Since \(p(1)=0\text{,}\) we have \(a_2+a_1+a_0=0\text{.}\) I claim \(p(x)=(-a_0)p_1+(-a_1)p_2\text{.}\) Indeed we have

\begin{equation*} -a_0p_1-a_1p_2=(-a_0-a_1)x^2+a_1x+a_0=a_2x^2+a_1x+a_0\text{,} \end{equation*}

since \(a_2+a_1+a_0=0\text{.}\) This shows that \(p(x)\in\Span\left(\{p_1, p_2\}\right)\text{,}\) as desired.
