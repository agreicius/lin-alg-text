More terminology:.

in the spirit of the last theorem, given a set of vectors \(S=\{\boldv_1,\dots, \boldv_r\}\text{,}\) we call \(W=\Span(S)\) the subspace of \(V\) generated by the vectors \(\boldv_i\text{.}\) Similarly, given a subspace \(W\text{,}\) a set \(S=\{\boldv_1,\dots, \boldv_r\}\) for which \(W=\Span(S)\) is called a spanning set of \(W\text{.}\)
