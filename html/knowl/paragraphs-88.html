Examples: \alert{the trivial vector space}.

Vector spaces of the form \(\R^n\) will be our main object of study. It is important to observe, however, that there are many other, more exotic vector spaces. In other words, you cannot always assume that a vector is an \(n\)-vector. We now introduce some of these more exotic examples, starting with the trivial vector space.

Let \(V=\{\boldv\}\) be a set containing exactly one element.

Addition and scalar multiplication.

Define:

\begin{align*} r\boldv\amp :=\amp \boldv \text{ for all \(r\in\R\) } ,\\ \boldv+\boldv\amp :=\amp \boldv \end{align*}
Additive identity and inverses.

Set:

\begin{align*} \boldzero\amp :=\amp \boldv\\ -\boldv\amp :=\amp \boldv\text{.} \end{align*}

It follows easily that \(V=\{\boldv\}\text{,}\) along with these operations and choices of zero vector and additive inverses, satisfies the definition of a vector space.

We call \(V\) the trivial (or zero) vector space, and denote it \(V=\{\boldzero\}\text{.}\) (Note that the single element of the trivial vector space is always the zero vector of the space. Hence the notation.)
