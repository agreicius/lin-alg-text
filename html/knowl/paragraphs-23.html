1.4: algebraic properties of matrices.

For the most part, the matrix operations of \(\cdot\) and \(+\) behave in much the same manner as their corresponding real number operations.

Theorem 1.5.1. Properties of matrix arithmetic.

The following properties hold for all matrices \(A, B, C\) and scalars \(a, b, c\) for which the given expression makes sense.

  1. \(\displaystyle A+B=B+A\)

  2. \(\displaystyle A+(B+C)=(A+B)+C\)

  3. \(\displaystyle A(BC)=(AB)C\)

  4. \(\displaystyle A(B+C)=AB+AC\)

  5. \(\displaystyle (B+C)A=BA+CA\)

  6. \(\displaystyle A(B-C)=AB-AC\)

  7. \(\displaystyle (B-C)A=BA-CA\)

  8. \(\displaystyle a(B+C)=aB+aC\)

  9. \(\displaystyle a(B-C)=aB-aC\)

  10. \(\displaystyle (a+b)C=aC+bC\)

  11. \(\displaystyle (a-b)C=aC-bC\)

  12. \(\displaystyle a(bC)=(ab)C\)

  13. \(a(BC)=(aB)C=B(aC)\text{.}\)

How does one actually prove one of these properties?

These are all matrix equalities. Using the definition of matrix equality, we must show (1) the LHS and RHS matrices are of the same size, and (2) that for each \((i,j)\text{,}\) the \((i,j)\)-th entry of the LHS is equal to the \((i,j)\)-th entry of the RHS. Let's see how this works on an example.
