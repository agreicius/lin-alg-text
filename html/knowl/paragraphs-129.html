Examples.

\alert{\(M_{mn}\)}. Define \(E_{ij}\) to be the matrix whose \((i,j)\)-th entry is 1, and whose every other entry is 0. Then the set

\begin{equation*} \{ E_{ij}\colon 1\leq i\leq m, 1\leq j\leq n\} \end{equation*}

is a spanning set for \(M_{mn}\text{.}\)

\(\R^n\).

In a similar vein, define \(\bolde_i\) to be the \(n\)-tuple whose \(i\)-th entry is 1, and whose every other entry is 0.

Then \(\{\bolde_1, \bolde_2, \dots, \bolde_n\}\) is a spanning set for \(\R^n\text{.}\)

\(P\) and \(P_n\).

The set \(\{1, x, x^2, \dots, \}\) is a spanning set for \(P\text{.}\) The set \(\{1, x, x^2, \dots, x^n\}\) is a spanning set for \(P_n\text{.}\)

\(\R^\infty\).

As above we can define \(\bolde_i\in \R^\infty\) to be the infinite sequence whose \(i\)-th entry is 1, and whose every other entry is \(0\text{.}\)

Note, however, that the set \(\{\bolde_1, \bolde_2, \bolde_3, \dots\}\) is not a spanning set for \(\R^\infty\text{.}\)

Indeed, the sequence \((1,1,1,\dots)\) is not a (finite!) linear combination of the \(\bolde_i\text{.}\)

\alert{\(\R_{>0}\)} Any \(a\ne 1\in \R_{>0}\) forms a spanning set for \(\R_{>0}\) as a vector space. This is because scalar multiplication by \(r\) in \(\R_{>0}\) is defined as exponentiation. Thus \(\Span(\{a\})=\{a^r\colon r\in\R\}=\R_{>0}\text{.}\) The last equality holds since the exponential function \(f(x)=a^x\) has range all positive reals for any base \(a\ne 1\text{.}\)
