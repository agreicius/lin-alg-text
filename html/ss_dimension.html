Section 3.6 Dimension

Chapter 3. Section 3.6: executive summary.
Definitions:.

dimension of a vector space. \alert{Procedures:} none, but many examples using “standard bases” for our familiar spaces. \alert{Theorems:} independent set can be extended to basis, spanning set can be contracted to basis; “street smarts” theorems; if \(W\subset V\text{,}\) then \(\dim W\leq \dim V\text{,}\) and \(\dim W=\dim V\) iff \(W=V\text{.}\)

Chapter 3. Section 3.6: dimension.
Theorem 3.6.1.

Suppose \(S=\{\boldv_1,\dots ,\boldv_n\}\) is a basis for the vector space \(V\text{.}\) Then every other basis \(S'\) for \(V\) contains exactly \(n\) elements. In fact:

  1. if \(S'\) contains {less} than \(n\) elements, then it does not span \(V\text{;}\)

  2. if \(S'\) contains {more} than \(n\) elements, then it is linearly dependent.

Proof.

To be sketched in class. Boil everything down to a system of linear equations.

Thanks to the theorem, the following notion of dimension is well-defined.

Definition 3.6.2.

Let \(V\) be a vector space. If \(V\) does not have a finite basis, then we say \(V\) is infinite dimensional and write \(\dim(V)=\infty\text{.}\) Otherwise we define the dimension of \(V\) to be the number \(n\) of vectors in a/any basis for \(V\text{,}\) and write \(\dim(V)=n\text{.}\)

Standard examples.

To prove \(\dim(V)=n\text{,}\) we must EXHIBIT a basis of \(V\) containing \(n\) elements.

  1. Since we have declared \(S=\{ \}\) to be a basis for \(\{\boldzero\}\text{,}\) we have \(\dim(\{\boldzero\})=0\text{.}\)

  2. Since the standard basis \(\{\bolde_1,\dots,\bolde_n\}\) of \(\R^n\) has \(n\) elements, we have \(\dim(\R^n)=n\text{.}\)

  3. Since the standard basis \(\{1,x,\dots ,x^n\}\) of \(P_n\) has \(n+1\) elements, we have \(\dim(P_n)=n+1\text{.}\)

  4. Since the standard basis \(\{E_{ij}\colon 1\leq i\leq m, 1\leq j\leq n\}\) of \(M_{mn}\) has \(mn\) elements, we have \(\dim(M_{mn})=mn\text{.}\)

To prove \(\dim(V)=\infty\text{,}\) we must prove \(V\) has no finite basis. We shown this already for \(P_\infty, C^\infty(-\infty,\infty), C^{(n)}(-\infty,\infty), C(-\infty,\infty), F(-\infty,\infty)\text{.}\)

Thus for each of these spaces \(V\) we have \(\dim(V)=\infty\text{.}\)

Street smarts.

For each pair \(V\) and \(S\) below, show super-super quickly that \(S\) is NOT a basis for \(V\text{.}\)

  1. \(V=M_{23}\text{.}\)

    \begin{equation*} S=\left\{ \begin{bmatrix}1\amp 1\amp 0\\0\amp 0\amp 0 \end{bmatrix} , \begin{bmatrix}0\amp 1\amp 1\\ 0\amp 0\amp 0 \end{bmatrix} , \begin{bmatrix}0\amp 0\amp 1\\ 1\amp 0\amp 0 \end{bmatrix} , \begin{bmatrix}0\amp 0\amp 0\\ 1\amp 1\amp 0 \end{bmatrix} , \begin{bmatrix}0\amp 0\amp 0\\ 0\amp 1\amp 1 \end{bmatrix} \right\} \end{equation*}

    Proof.

    We have 5 vectors living in a 6-dimensional space. By Theorem 3.6.1, these vectors cannot span.

  2. \(V=P_3\text{,}\)

    \begin{equation*} S=\{1+\pi x-x^3, 1+\sqrt{e}x^2+x^3, 1+x+x^2+x^3, 1-x^3, x+17x^2+x^3\}\text{.} \end{equation*}

    Proof.

    We have 5 vectors living in a 4-dimensional space. By Theorem 3.6.1 they must be linearly dependent.

Here are some more useful facts about bases, the proofs of which will be done in class and the exercises.

Theorem 3.6.3.

Let \(V\) be finite dimensional.

  1. If \(S\) spans \(V\text{,}\) then some subset of \(S\) is a basis of \(V\text{.}\) (Basically, throw out the redundant members!).

  2. If \(S\) is linearly independent, then \(S\) can be extended to a basis of \(V\text{.}\) (Basically, add vectors to \(S\) one by one until it spans.)

  3. Suppose \(\dim V=n\text{,}\) and let \(S=\{\boldv_1,\dots ,\boldv_n\}\) be any set of \(n\) vectors in \(V\text{.}\) Then

    \begin{equation*} S \text{ spans \(V\) } \Leftrightarrow S \text{ is linearly independent }\text{.} \end{equation*}

    Thus if we know beforehand that \(\dim V=n\text{,}\) then to show a given set \(S=\{\boldv_1,\dots ,\boldv_n\}\) is a basis it suffices to show either that is spans, or that it is linearly independent.

Dimension can be a useful tool for deciding whether a subspace \(W\) of \(V\) is in fact all of \(V\text{.}\)

Theorem 3.6.4.

Suppose \(\dim(V)=n\text{,}\) and let \(W\subset V\) be a subspace of \(V\text{.}\)

  1. \(\dim(W)\leq \dim(V)\text{.}\)

  2. \(\dim(W)=\dim(V)\) if and only if \(W=V\text{.}\)

Proof.

First we show \(W\) is finite-dimensional. Indeed, if not, we could find an infinite set \(S\) of linearly independent vectors, contradicting the fact that in an \(n\)-dimensional space there can be at most \(n\) linearly independent vectors.

Thus we have \(\dim(W)=r\) for some finite \(r\text{,}\) which means we can find a basis \(S=\{\boldv_1,\dots, \boldv_r\}\) of \(W\text{.}\) Since these are linearly independent vectors in \(V\text{,}\) by the previous theorem we can extend \(S\) to a basis \(\{\boldv_1,\dots, \boldv_r,\boldv_{r+1},\dots,\boldv_n\}\) of \(V\text{,}\) showing \(r\leq n\text{,}\) and thus \(\dim(W)\leq\dim(V)\text{.}\)

Example.

Describe all subspaces of \(\R^3\text{.}\)

Since \(\dim(\R^3)=3\text{,}\) by the previous theorem a subspace \(W\subset \R^3\) has dimension 0, 1, 2, or 3.

\(\dim(W)=0\text{.}\).

In this case \(W=\{\boldzero\}\) is the trivial subspace.

\(\dim(W)=1\text{.}\).

In this case \(W=\Span\{\boldv\}\) of a single nonzero vector. We recognize this as a line in 3-space.

\(\dim(W)=2\text{.}\).

In this case \(W=\Span\{\boldv, \boldw\}\) is the span of two linearly independent vectors. This means that they are not scalar multiples of one another, hence not colinear. It follows that \(W\) is a plane in 3-space.

\(\dim(W)=3\text{.}\).

Since \(W\subset\R^3\) and \(\dim(\R^3)=3\text{,}\) it follows from the previous theorem that \(W=\R^3\text{.}\)

We have shown that the only subspaces of \(\R^3\) are lines, planes, the origin, and \(\R^3\) itself.

Dimension theorem compendium.

We collect here some of the most important results from this section.

Theorem 3.6.5.

{Dimension theorem compendium} Let \(V\) be a vector space, and suppose \(B=\{\boldv_1,\boldv_2,\dots, \boldv_n\}\) is a basis for \(V\text{.}\)

  1. (Street smarts) If \(S\) is a subset of \(V\) containing less than \(n\) elements, then \(S\) does not span \(V\text{.}\)

  2. (Street smarts) If \(S\) is a subset of \(V\) containing more than \(n\) elements, then \(S\) is linearly dependent.

  3. All bases of \(V\) contain exactly \(n\) elements.

  4. If \(S\) is linearly independent, then \(S\) can be extended to a basis of \(V\text{.}\)

  5. If \(S\) spans \(V\text{,}\) then some subset of \(S\) is a basis for \(V\text{:}\) i.e., \(S\) can be contracted to a basis.

  6. If \(\val{S}=n\text{,}\) then \(S\) is a basis if and only if \(S\) is linearly independent if and only if \(S\) spans \(V\text{.}\)

  7. (Dimension of subspaces I) Given subspace \(W\subseteq V\text{,}\) we have \(\dim W\leq \dim V=n\text{.}\)

  8. (Dimension of subspaces II) Given subspace \(W\subseteq V\text{,}\) we have

    \begin{equation*} W=V \text{ if and only if } \dim W=\dim V\text{.} \end{equation*}
