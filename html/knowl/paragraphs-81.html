Definition contd..

  1. \(\boldv+\boldw=\boldw+\boldv\) (addition is commutative)

  2. \(\boldu+(\boldv+\boldw)=(\boldu+\boldv)+\boldw\) (addition is associative)

  3. There is an element \(\boldzero\in V\) satisfying \(\boldzero+\boldv=\boldv\) for all \(\boldv\in V\text{.}\) (\(V\) has an additive identity)

  4. For all \(\boldv\in V\text{,}\) there is an element \(-\boldv\in V\) s.t. \(\boldv+(-\boldv)=\boldzero\text{.}\) (\(V\) has additive inverses)

  5. \(r(\boldu+\boldv)=r\boldu+r\boldv\) (scal. mult. distributes over vector sums)

  6. \((r+s)\boldv=r\boldv+s\boldv\) (sum of scalars distributes)

  7. \(r(s\boldv)=(rs)\boldv\) (scal. mult. is assoc.)

  8. \(1\boldv=\boldv\) for all \(\boldv\in V\) (scalar 1 acts as identity)

Elements of a vector space \(V\) are called vectors.

A linear combination in a vector space is any expression of the form \(c_1\boldv_1+c_2\boldv_2+\cdots +c_r\boldv_r\text{,}\) where \(c_i\in\R\) and \(\boldv_i\in V\) for all \(1\leq i\leq r\text{.}\)
