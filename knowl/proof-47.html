Proof

Let \(B=\{\boldv_1,\dots, \boldv_n\}\) and let \(B'=\{\boldw_1,\dots,\boldw_n\}\text{.}\)

We build \(\underset{B\rightarrow B'}{P}\) \alert{column by column}: specifically, \(\underset{B\rightarrow B'}{P}\) is the matrix whose \(j\)-th column is

\begin{equation*} \boldp_j=[\boldv_j]_{B'}; \end{equation*}

i.e., to compute the \(j\)-th column of \(\underset{B\rightarrow B'}{P}\) you must find the \(B'\)-coordinates of the \(j\)-th basis element of \(B\text{.}\)

Thus defined, one can show using the column method that \(\underset{B\rightarrow B'}{P}[\boldv_j]_{B}=[\boldv_j]_{B'}\text{.}\) One then argues that if the condition is true for each basis element \(\boldv_j\text{,}\) then it must be true for all \(\boldv\text{.}\) (Ask your professor for details!)
