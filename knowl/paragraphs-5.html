Linear systems.
Definition 1.1.2.

A system of linear equations (or linear system) is a set of linear equations.

We display a system of \(m\) equations in the \(n\) unknowns \(x_1,x_2,\dots x_n\) as follows:

\begin{equation*} \eqsys \end{equation*}

A homogenous linear system is one where \(b_i=0\) for all \(i\text{:}\)

\begin{equation*} \homsys \end{equation*}
Comment..

It will be helpful to get comfortable to the double-indexed constants \(a_{ij}\) as soon as possible. Here is a good way to understand this:

  • \(i\text{,}\) the first index, indicates the \(i\)-th {row}, or equivalently, the \(i\)-th equation;

  • \(j\text{,}\) the second index, indicates the \(j\)-th {column}, which is associated to the \(j\)-th variable.
