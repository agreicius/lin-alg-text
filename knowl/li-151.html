Item 1.

Let \(V=\{\boldzero\}\text{.}\) The only subsets of \(V\) are the empty set \(S=\{ \}\text{,}\) and the set \(S=\{\boldzero\}\text{.}\) The first set doesn't span \(V\text{;}\) the second set is not linearly independent. So technically the zero space has no basis! This turns out to be a significant inconvenience, so by executive order we declare the empty set \(B=\{ \}\) to be a basis of the zero space \(V=\{\boldzero\}\text{.}\)
