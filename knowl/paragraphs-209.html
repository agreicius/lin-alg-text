Coordinate vector map.
Theorem 4.3.3.

Let \(V\) be a vector space, and suppose \(B=\{\boldv_1,\boldv_2,\dots, \boldv_n\}\) be a basis for \(V\text{.}\) Define

\begin{align*} T\colon V\amp \rightarrow \R^n\\ \boldv\amp \mapsto [\boldv]_B\text{.} \end{align*}

  1. \(T\) is a linear transformation. We will call \(T\) the coordinate vector map with respect to \(B\).

  2. \(T(\boldv_1)=T(\boldv_2)\) if and only if \(\boldv_1=\boldv_2\text{:}\) i.e., \(T\) is one-to-one. In particular, \(T(\boldv)=\boldzero\) if and only if \(\boldv=\boldzero_V\text{.}\)

  3. \(\range T=\R^n\text{:}\) i.e. \(T\) is onto.

  4. A set \(S=\{w_1,w_2,\dots, w_r\}\subseteq V\) is linear independent if and only if \(T(S)=\{T(\boldw_1), T(\boldw_2),\dots, T(\boldw_r)\}\subseteq \R^n\) is linearly independent.

  5. A set \(S=\{w_1,w_2,\dots, w_r\}\subseteq V\) spans \(V\) if and only if \(T(S)=\{T(\boldw_1), T(\boldw_2),\dots, T(\boldw_r)\}\) spans \(\R^n\text{.}\)
