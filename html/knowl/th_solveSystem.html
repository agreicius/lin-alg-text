Theorem 1.3.5. Solving linear systems algorithm.

Let \(L\) be a linear system in the unknowns \(x_1,x_2,\dots, x_n\text{.}\) The set of solutions to \(L\) can be computed as follows.

Step 1

Apply Gaussian elimination to reduce \(L\) to an equivalent system \(L'\) in row echelon form.

Step 2

Let \(U\) be the augmented matrix associated to \(L'\text{.}\) If the last column of \(U\) has a leading one, then \(L\) is inconsistent, and the set of solutions is the empty set \(\emptyset=\{\ \}\text{.}\) Otherwise, proceed to the next step.

Step 3

Determine which if any of the unknowns are free variables of \(L'\text{.}\)

Step 4

If there are no free variables, solve for each unknown using back-substitituion. This is the unique solution to the system.

Otherwise, let \(x_{i_1}, x_{i_2},\dots , x_{i_r}\) be the free variables of \(L'\text{,}\) and let \(x_{j_1},x_{j_2},\dots, x_{j_s}\) be the leading variables. Bringing all free variables to the right of the equals signs in the system and using back substitution we can solve for each leading variable as a function of the free variables: i.e., we have

\begin{equation*} x_{j_1}=F_1(x_{i_1},x_{i_2},\dots, x_{i_r}),\ x_{j_2}=F_2(x_{i_1},x_{i_2},\dots, x_{i_r}),\dots, \ x_{j_s}=F_s(x_{i_1},x_{i_2},\dots, x_{i_r})\text{,} \end{equation*}

where the \(F_i\) are certain functions of \(r\) variables. It follows that the set of solutions to \(L\) has the parametric description

\begin{align*} \text{Free variables:}\amp \amp x_{i_1}\amp=t_{1}, \ x_{i_2}=t_{2}, \ \dots, \ x_{i_r}=t_{r}\\ \text{Leading variables:}\amp \amp x_{j_1}\amp =F_1(t_1,t_2,\dots, t_r),\ x_{j_2} =F_2(t_1,t_2,\dots, t_r) ,\ \dots,\ x_{j_s} =F_s(t_1,t_2,\dots, t_r) \text{,} \end{align*}

where \(t_1,t_2,\dots, t_n\) are any real numbers.

Proof.
