Section 4.3 Coordinate vectors

Coordinate vectors relative to a basis $B$.
Definition 4.3.1.

Let \(B=\{\boldv_1,\dots ,\boldv_n\}\) be a basis for \(V\text{,}\) and let \(\boldv\in V\text{.}\) According to Theorem 3.5.2 there is a unique choice of \(c_i\) such that

\begin{equation*} \boldv=c_1\boldv_1+c_2\boldv_2\cdots +c_n\boldv_n\text{.} \end{equation*}

We define the coordinate vector of \(\boldv\) relative to the basis \(B\) to be the \(n\)-tuple

\begin{equation*} [\boldv]_B:=(c_1,c_2,\dots, c_n) \end{equation*}
Remark 4.3.2.

  1. To compute the coordinate vector of \(\boldv\) relative to \(B=\{\boldv_1,\boldv_2,\dots, \boldv_n\}\) we must solve the vector equation \(\boldv=c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n\) for the coefficients \(c_1, c_2,\dots, c_n\text{.}\) For any given example this computation can usually be reduced to setting up and solving a certain system of linear equations.

  2. As usual, we will be flexible in terms of how we treat the \(n\)-vector \((c_1,c_2,\dots, c_n)\text{.}\) For example, we will often interpret this as a \(n\times 1\) column vector.

Example.

Let \(V=P_2\text{.}\)

  1. Let \(B=\{1,x, x^2\}\) be the standard basis of \(V\text{.}\) Computing \([p(x)]_B\) in this case is easy since any polynomial \(p(x)=a+bx+cx^2\) is already expressed as a linear combination of the elements of \(B\) !! That is, we have \([a+bx+cx^2]_B=(a,b,c)\) for any polynomial \(p(x)=a+bx+cx^2\text{.}\) Some explicit examples:

    \begin{align*} _B\amp = (7, -\pi, \sqrt{2})\\ [1+9x^2]_B\amp =(1,0,9) \end{align*}

  2. Let \(B'=\{1+x+x^2, -x+x^2, -1+x^2\}=\{p_1,p_2,p_3\}\) be the nonstandard basis we considered earlier. Now it requires more work to compute \([p(x)]_{B'}\text{.}\) Some explicit examples:

    \begin{align*} 1+x+x^2=1p_1+0p_2+0p_3\amp \Rightarrow\amp [1+x+x^2]_{B'}=(1,0,0)\\ -x+x^2=0p_1+1p_2+0p_3\amp \Rightarrow\amp [-x+x^2]_{B'}=(0,1,0)\\ x^2=\frac{1}{3}p_1+\frac{1}{3}p_2+\frac{1}{3}p_3\amp \Rightarrow\amp [x^2]_{B'}=(\frac{1}{3},\frac{1}{3},\frac{1}{3}) \end{align*}

The last example was computed by setting up the polynomial equation \(x^2=c_1(1+x+x^2)+c_2(-x+x^2)+c_3(-1+x^2)\) and solving for the \(c_i\text{.}\) Verify for yourself that \(c_1=c_2=c_3=1/3\) is the solution!

Coordinate vector map.
Theorem 4.3.3.

Let \(V\) be a vector space, and suppose \(B=\{\boldv_1,\boldv_2,\dots, \boldv_n\}\) be a basis for \(V\text{.}\) Define

\begin{align*} T\colon V\amp \rightarrow \R^n\\ \boldv\amp \mapsto [\boldv]_B\text{.} \end{align*}

  1. \(T\) is a linear transformation. We will call \(T\) the coordinate vector map with respect to \(B\).

  2. \(T(\boldv_1)=T(\boldv_2)\) if and only if \(\boldv_1=\boldv_2\text{:}\) i.e., \(T\) is one-to-one. In particular, \(T(\boldv)=\boldzero\) if and only if \(\boldv=\boldzero_V\text{.}\)

  3. \(\range T=\R^n\text{:}\) i.e. \(T\) is onto.

  4. A set \(S=\{w_1,w_2,\dots, w_r\}\subseteq V\) is linear independent if and only if \(T(S)=\{T(\boldw_1), T(\boldw_2),\dots, T(\boldw_r)\}\subseteq \R^n\) is linearly independent.

  5. A set \(S=\{w_1,w_2,\dots, w_r\}\subseteq V\) spans \(V\) if and only if \(T(S)=\{T(\boldw_1), T(\boldw_2),\dots, T(\boldw_r)\}\) spans \(\R^n\text{.}\)

Proof.

Exercise.

Fix vector space \(V\) and basis \(B=\{\boldv_1,\boldv_2,\dots, \boldv_n\}\text{.}\) According to Theorem 4.3.3, the coordinate vector map \(\boldv\mapsto [\boldv]_B\) is not only linear, it is also 1-1 and onto. Such functions are called isomorphisms.

Definition 4.3.4.

Let \(V\) and \(W\) be vector spaces. An isomorphism is a linear transformation \(T\colon V\rightarrow W\) that is 1-1 and onto.

The spaces \(V\) and \(W\) are called isomorphic in this case.

We apply the general principle described in the previous slide to the special case of a coordinate vector isomorphism \([\hspace{10pt}]_B\colon V\rightarrow \R^n\text{.}\)

Coordinate vector technique.

Suppose we have chosen a finite basis \(B\) for the vector space \(V\text{.}\) Let \([\hspace{10pt}]_B\) denote the corresponding coordinate vector map, and let \([\hspace{10pt}]_B^{-1}\) denote the inverse of this map.

Any linear algebraic questions we want to answer about \(V\) (involving span, linear independence, etc.) can be answered as follows:

  1. First apply \([\hspace{10pt}]_B\) to all vectors \(\boldv\) in question.

  2. Answer the corresponding question in \(\R^n\) about the column vectors \([\boldv]_B\text{.}\)

  3. If necessary, translate your results back to the setting of \(V\) by applying the inverse \([\hspace{10pt}]_B^{-1}\text{.}\)

Example.

The set

\begin{equation*} S=\left \{ A_1=\begin{bmatrix}2\amp 1\\ 0\amp -2 \end{bmatrix} , A_2=\begin{bmatrix}1\amp 1\\ 1\amp -1 \end{bmatrix} , A_3=\begin{bmatrix}0\amp 1\\ 2\amp 0 \end{bmatrix} , A_4=\begin{bmatrix}-1\amp 0\\ 1\amp 1 \end{bmatrix} \right\} \end{equation*}

is a subset of the space \(W=\{ A\in M_{22}\colon \tr A=0\}\text{.}\) (Recall: the trace of a matrix, \(\tr A\text{,}\) is defined as the sum of its diagonal elements. )

Contract \(S\) to a basis of \(\Span S\text{,}\) and determine whether \(\Span S=W\text{.}\) Use the coordinate vector map technique!

We must first pick a basis for \(W\text{.}\) Observe that the general element of of \(W\) can be described as

\begin{equation*} \begin{bmatrix}a\amp b\\ c\amp -a \end{bmatrix} =a\underset{B_1}{\underbrace{\begin{bmatrix}1\amp 0\\ 0\amp -1 \end{bmatrix} }}+b\underset{B_2}{\underbrace{\begin{bmatrix}0\amp 1\\ 0\amp 0 \end{bmatrix} }}+c\underset{B_3}{\underbrace{\begin{bmatrix}0\amp 0\\1\amp 0 \end{bmatrix} }}\text{.} \end{equation*}

It follows easily that \(B=\{B_1,B_2, B_3\}\) is a basis for \(W\text{.}\)

Apply \([\hspace{10pt}]_B\) to the elements of the given \(S\) to get a corresponding set \(S'\subseteq\R^3\text{:}\)

\begin{equation*} S'=\left\{ [A_1]_B=(2,1,0), [A_2]_B=(1,1,1), [A_3]_B=(0,1,2), [A_4]_B=(-1,0,1)\right\} \end{equation*}

Now use the “contract to basis” algorithm on \(S'\) to conclude that the set \(\{(2,1,0), (1,1,1)\}\) forms a basis for \(\Span S'\text{.}\) Translating back to \(W\text{,}\) we see that \(A_1\) and \(A_2\) form a basis for \(\Span S\text{,}\) that \(\dim\Span S=2\text{,}\) and hence that \(\Span S\ne W\text{,}\) since \(\dim W=3\text{.}\)
