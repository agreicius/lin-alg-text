Some comments about how we express the solutions to a system. I described the general solution in tuple form:

\begin{equation*} (x_1,x_2,\dots,x_5)=(5-2s-3t, s, 1, t, 2), \ \text{ \(s, t\) any real numbers }\text{.} \end{equation*}

Alternatively we can give parametric equations for each variable separately:

\begin{equation*} \begin{array}{lcl} x_1=7-2s-3t\\ x_2=s\\ x_3=1\amp \amp s,t\in\R \\ x_4=t\\ x_5=2\amp \amp \end{array} \end{equation*}

Of course we can also just write down the set of all solutions:

\begin{equation*} S=\Bigl\{(7-2s-3t, s, 1, t, 2) \colon s,t\in\R\Bigr\}\text{.} \end{equation*}

Get used to using all three forms of answers.
