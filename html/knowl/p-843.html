To compute the coordinate vector of \(\boldv\) relative to \(B=\{\boldv_1,\boldv_2,\dots, \boldv_n\}\) we must solve the vector equation \(\boldv=c_1\boldv_1+c_2\boldv_2+\cdots +c_n\boldv_n\) for the coefficients \(c_1, c_2,\dots, c_n\text{.}\) For any given example this computation can usually be reduced to setting up and solving a certain system of linear equations.
