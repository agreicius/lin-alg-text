Examples: \alert{$C^\infty(X)$}.

Let \(X\subset\R\) be any interval of any sort: e.g. \(X=[a,b]\text{,}\) \(X=(-\infty, \infty)\text{,}\) \(X=(-\infty, b]\text{,}\) etc. Define \(V=C^\infty(X)\) to be the set of all infinitely-differentiable real-valued functions with domain \(X\text{.}\)

Addition and scalar multiplication.

Given any elements \(f, g\in C^\infty(X)\text{,}\) define:

\begin{align*} rf\amp :=\amp \text{ the function defined as \((rf)(x)=rf(x)\) for all \(r\in\R\) } ,\\ f+g\amp :=\amp \text{ the function defined as \((f+g)(x)=f(x)+g(x)\). } \end{align*}

In other words, vector scalar multiplication is defined to be scalar multiplication of functions, and vector addition is defined to be function addition.

Additive identity and inverses.

Set:

\begin{align*} \boldzero\amp :=\amp \text{ the zero function \(0_X\), defined as \(0_X(x)=0\) for all \(x\in X\). }\\ -f\amp :=\amp \text{ the function defined as \((-f)(x)=-f(x)\) for all \(x\in X\) } \text{.} \end{align*}

The fact that \(C^\infty(X)\) with these operations forms a vector space follows from familiar properties of function arithmetic.

Take a moment to let the exotic quality of this example sink in. Our “vectors” in this example are functions! Whereas we can concretely describe the general element of \(\R^3\) (\((a,b,c)\)) or even \(\R^n\) (\((r_1,r_2,\dots, r_n)\)), this is not so in \(C^\infty(X)\text{:}\) the general element is just some infinitely differentiable function \(f\text{.}\) I can give you plenty of examples of elements (e.g., \(f(x)=x^2\text{,}\) \(f(x)=\sin x-e^x\text{,}\) etc.), but I cannot express the general element in any kind of finite way.
