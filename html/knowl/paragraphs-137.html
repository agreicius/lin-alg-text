Null space of a linear transformation.

The notion of span is a useful tool for constructing subspaces inside a space \(V\text{:}\) given any collection \(\{\boldv_1, \boldv_2, \dots, \boldv_r\}\text{,}\) we now know that the set \(W=\Span(\{\boldv_1,\boldv_2,\dots, \boldv_r\})\) is guaranteed to be a subspace of \(V\text{.}\)

The notion of a null space of a linear transformation \(T\colon V\rightarrow W\) provides another useful tool for constructing subspaces.

Definition 3.3.5.

The null space of a linear transformation \(T\colon V\rightarrow W\) is the set

\begin{equation*} \NS T\colon =\{\boldv\colon T(\boldv)=\boldzero_W \}\text{.} \end{equation*}

In the special case where \(T\colon \R^n\rightarrow \R^m\) and \(T=T_A\text{,}\) we may write \(\NS A\) for \(\NS T_A\text{:}\) by definition this is the set

\begin{equation*} \NS A=\NS T_A=\{\boldx\in \R^n\colon A\boldx=\underset{m\times 1}{\boldzero}\}\text{.} \end{equation*}

In other words, \(\NS A\) is the set of solutions to the homogenous matrix equation \(A\boldx=\boldzero\text{,}\) or equivalently, its associated homogenous system of equations.

As we see in the next theorem, given linear \(T\colon V\rightarrow W\text{,}\) its null space \(\NS T\) is always a subspace !!
