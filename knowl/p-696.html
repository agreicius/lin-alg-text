A set \(B=\{\boldv_1,\dots, \boldv_n\}\) is a basis of \(V\) if and only if every vector \(\boldv\in V\) can be written as a linear combination of \(\boldv_i\) in a unique way: i.e., for every \(\boldv\in V\) there is exactly one choice of scalars \(c_1,c_2, \dots c_n\) such that

\begin{equation*} \boldv=c_1\boldv_1+c_2\boldv_2\cdots +c_n\boldv_n\text{.} \end{equation*}
