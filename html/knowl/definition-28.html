Definition 2.1.3.

Let \(A=[a_{ij}]_{n\times n}\text{.}\)

Expansion along row..

Pick any \(i\text{.}\) Then we define

\begin{equation*} \det(A)=\sum_{j=1}^na_{ij}C_{ij}=a_{i1}C_{i1}+a_{i2}C_{i2}\dots +a_{in}C_{in} \end{equation*}
Expansion along column..

Pick any \(j\text{.}\) Then we define

\begin{equation*} \det(A)=\sum_{i=1}^na_{ij}C_{ij}=a_{1j}C_{1j}+a_{2j}C_{2j}\dots +a_{nj}C_{nj}\text{.} \end{equation*}
