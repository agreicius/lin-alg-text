Definition 1.4.10.

Let \(A\) be any \(n\times n\) matrix. For any integer \(r\geq 0\) we define the \(r\)-th power of \(A\text{,}\) \(A^{r}\text{,}\) as follows:

  1. \(A^0=I_n\text{.}\)

  2. \(\ds A^r=\underbrace{A\cdot A\cdots A}_{r \text{ times } }\) for \(r>0\text{.}\)

Furthermore, if \(A\) is invertible, we define

\begin{equation*} A^{-r}=\left(A^{-1}\right)^r\text{,} \end{equation*}

for any positive integer \(r>0\text{.}\)
