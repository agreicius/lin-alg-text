Definitions:.

dimension of a vector space. \alert{Procedures:} none, but many examples using “standard bases” for our familiar spaces. \alert{Theorems:} independent set can be extended to basis, spanning set can be contracted to basis; “street smarts” theorems; if \(W\subset V\text{,}\) then \(\dim W\leq \dim V\text{,}\) and \(\dim W=\dim V\) iff \(W=V\text{.}\)
