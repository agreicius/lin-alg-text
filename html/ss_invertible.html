Section 1.7 More on invertibility

Chapter 1: Section 1.7: executive summary.
Definitions:.

none. \alert{Procedures:} deciding when a linear system \(A\boldx=\boldb\) is consistent. \alert{Theorems:} invertibility theorem (expanded), \(AB\) invertible if and only if \(A\) and \(B\) are invertible, \(A\) is a left inverse of \(B\) if and only if \(A\) is a right inverse of \(B\text{.}\)

Chapter 1: Section 1.7: expanded invertibility theorem.

We add two more equivalent statements of invertibility to our invertibility theorem (IT).

Theorem 1.7.1. Expanded invertibility theorem.

Let \(A=[a_{ij}]\) be a square \(n\times n\) matrix. The following statements are equivalent.

  1. \(A\) is invertible.

  2. \(A\boldx=\boldzero\) has a unique solution (the trivial one).

  3. \(A\) is row equivalent to \(I_n\text{,}\) the \(n\times n\) identity matrix.

  4. \(A\) is a product of elementary matrices.

  5. \(A\boldx=\boldb\) has a unique solution for every \(n\times 1\) column vector \(\boldb\text{.}\)

  6. \(A\boldx=\boldb\) has a solution for every \(n\times 1\) column vector \(\boldb\text{.}\)

Proof.

We already know statements (a)-(d) are equivalent. Adding (e) and (f) to this list is left as an exercise.

Corollary 1.7.2.

Let \(A\) and \(B\) be \(n\times n\text{.}\) Then \(AB\) is invertible if and only if \(A\) and \(B\) are both invertible.

Proof.

We have already proved the \(\Leftarrow\) direction of this equivalnce.

For the \(\Rightarrow\) direction, assume \(AB\) is invertible and let \(C\) be its inverse, so that \(C(AB)=(AB)C=I_n\text{.}\)

We first prove \(B\) is invertible, using equivalent statement (b) of IT; that is we will prove the implication \(B\boldx=\boldzero\Rightarrow \boldx=\boldzero\text{.}\)

\begin{align*} B\boldx=\boldzero\amp \Rightarrow \amp AB\boldx=\boldzero\\ \amp \Rightarrow\amp \boldx=\boldzero \ \text{ (since \(AB\) invertible) } \text{.} \end{align*}

This proves \(B\) is invertible, and hence that \(B^{-1}\) exists.

Next we prove \(A\) is invertible, by explicitly exhibiting an inverse: namely, \(A^{-1}=BC\text{.}\)

Indeed we have \(A(BC)=(AB)C=I_n\text{,}\) from above. For the other direction we have

\begin{align*} C(AB)=I\amp \Leftrightarrow\amp CA=B^{-1} \ \text{ (mult. both sides on right by \(B^{-1}\))}\\ \amp \Leftrightarrow\amp (BC)A=I_n \text{ (mult. both sides on left by \(B\)) } \text{.} \end{align*}

This proves \(A\) and \(B\) are both invertible, and completes our proof.

Corollary 1.7.3.

Let \(A\) and \(B\) be \(n\times n\text{.}\)

  1. \(BA=I_n\Rightarrow AB=I_n\text{.}\)

  2. \(AB=I_n\Rightarrow BA=I_n\text{.}\)

In other words, to show \(B\) is the inverse of \(A\) it is enough to show either that it is a left-inverse (\(BA=I_n\)), or a right-inverse (\(AB=I_n\)).

Proof.

It is enough to prove the first implication: the second then follows by exchanging the roles of \(A\) and \(B\text{!}\)

Suppose \(BA=I_n\text{.}\) I will first show that \(A\) is invertible. Indeed, suppose \(A\boldx=\boldzero\text{.}\) Then \(BA\boldx=\boldzero\text{.}\) But \(BA=I_n\text{,}\) and \(I\boldx=\boldzero\) implies \(\boldx=\boldzero\text{.}\) It follows from statement (b) of IT that \(A\) is invertible.

Now that we know \(A^{-1}\) exists we have

\begin{align*} BA=I_n\amp \Rightarrow \amp B=A^{-1} \ \text{ (mult. both sides on right by \(A^{-1}\))}\\ \amp \Rightarrow \amp AB=I_n \ \text{ (mult. both sides on left by \(A\)) } \end{align*}

Question:.

what conditions on \(A\) and \(\boldb\) guarantee the system is consistent; equivalently, when we can solve the matrix equation above?

Partial answer:.

the preceding discussion gives us a partial answer. If \(A\) happens to be an invertible matrix (in particular, it must be square!), then the equation is guaranteed to have a (unique) solution. We just solve for \(\boldx\) directly in this case: \(\boldx=A^{-1}\boldb\text{.}\)

Otherwise:.

if \(A\) is not invertible, then we resort to the theory of Gaussian elimination to answer the question. Take the augmented matrix \([A \vert \boldb]\text{,}\) row reduce to row echelon form \([U\vert \boldb']\) and reason from there. The example that follows illustrates this.

Diagonal, triangular and symmetric matrices.

Lastly, we officially introduce some special families of square matrices, as well as a corresponding invertibility theorem. The proof of the latter will be done in class.

Definition 1.7.4.

Let \(A=[a_{ij}]\) be \(n\times n\text{.}\) We say

  1. \(A\) is diagonal if \(a_{ij}=0\) for all \((i,j)\) with \(i\ne j\text{.}\) ( “\(A\) is zero off the diagonal.” )

  2. \(A\) is upper triangular if \(a_{ij}=0\) for all \((i,j)\) with \(j>i\text{.}\) ( “\(A\) is zero below the diagonal.” )

  3. \(A\) is lower triangular if \(a_{ij}=0\) for all \((i,j)\) with \(i>j\text{.}\) ( “\(A\) is zero above the diagonal.” )

  4. \(A\) is triangular if \(A\) is upper triangular or lower triangular.

  5. \(A\) is symmetric if \(A^T=A\text{.}\) (Equivalently, \(a_{ij}=a_{ji}\) for all \(1\leq i,j\leq n\text{.}\))

Theorem 1.7.5. Invertibility of triangular matrices.

Let \(A=[a_{ij}]\) be a triangular \(n\times n\) matrix. Then \(A\) is invertible if and only if \(a_{ii}\ne 0\) for all \(1\leq i\leq n\text{.}\)

In other words, \(A\) is invertible if and only if the diagonal entries of \(A\) are all nonzero.
