Examples: \alert{$\R^\infty$}.
Definition 3.1.2.

We define \(\R^\infty\) to be the set of all infinite sequences of real numbers: i.e.,

\begin{equation*} \R^\infty=\{(t_1,t_2,t_3,\dots)\colon t_i\in\R \}\text{.} \end{equation*}

Define a vector space structure on \(\R^\infty\) as follows.

Addition and scalar multiplication.

Given any \(\boldv=(v_1,v_2,\dots)\) and \(\boldw=(w_1,w_2, \dots )\) in \(\R^n\) define:

\begin{align*} r\boldv\amp :=\amp (rv_1,rv_2,\dots) \text{ for all \(r\in\R\) } ,\\ \boldv+\boldw\amp :=\amp (v_1+w_1,v_2+w_2, \dots) \end{align*}
Additive identity and inverses.

Set:

\begin{align*} \boldzero\amp :=\amp (0,0,\dots)\\ -\boldv\amp :=\amp (-v_1,-v_2,\dots)\text{.} \end{align*}

It is easy to show \(\R^\infty\) with these operations forms a vector space.
