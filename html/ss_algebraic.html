Section 1.5 Algebraic properties

Chapter 1. Section 1.5: executive summary.
Definitions.

: zero matrix \(\boldzero_{m\times n}\text{,}\) identity matrix \(I_n\text{,}\) inverse matrix \(A^{-1}\text{,}\) invertibility, \(A^r\) and \(A^{-r}\) for nonnegative integer \(r\geq 0\text{.}\) \alert{Procedures}: none! \alert{Theorems}: Many! Lots of familiar rules of real number algebra (associativity, distributivity, etc.) also hold for matrix algebra. However some important properties fail: e.g., commutativity and multiplicative cancellation. We also learn some properties about taking the transpose (\(A\mapsto A^T\)) and its relation with taking the inverse (\(A\mapsto A^{-1}\)).

1.4: algebraic properties of matrices.

For the most part, the matrix operations of \(\cdot\) and \(+\) behave in much the same manner as their corresponding real number operations.

Theorem 1.5.1. Properties of matrix arithmetic.

The following properties hold for all matrices \(A, B, C\) and scalars \(a, b, c\) for which the given expression makes sense.

  1. \(\displaystyle A+B=B+A\)

  2. \(\displaystyle A+(B+C)=(A+B)+C\)

  3. \(\displaystyle A(BC)=(AB)C\)

  4. \(\displaystyle A(B+C)=AB+AC\)

  5. \(\displaystyle (B+C)A=BA+CA\)

  6. \(\displaystyle A(B-C)=AB-AC\)

  7. \(\displaystyle (B-C)A=BA-CA\)

  8. \(\displaystyle a(B+C)=aB+aC\)

  9. \(\displaystyle a(B-C)=aB-aC\)

  10. \(\displaystyle (a+b)C=aC+bC\)

  11. \(\displaystyle (a-b)C=aC-bC\)

  12. \(\displaystyle a(bC)=(ab)C\)

  13. \(a(BC)=(aB)C=B(aC)\text{.}\)

How does one actually prove one of these properties?

These are all matrix equalities. Using the definition of matrix equality, we must show (1) the LHS and RHS matrices are of the same size, and (2) that for each \((i,j)\text{,}\) the \((i,j)\)-th entry of the LHS is equal to the \((i,j)\)-th entry of the RHS. Let's see how this works on an example.

Proof.

Let \(A=[a_{ij}]_{m\times r}\text{,}\) \(B=[b_{ij}]_{r\times s}\text{,}\) \(C=[c_{ij}]_{s\times n}\text{.}\) Then

\begin{equation*} A(BC)=(AB)C\text{.} \end{equation*}

(1) One checks easily that the matrices on the LHS and RHS are both of size \(m\times n\text{,}\) so we move straight to (2) using our index notation:

\begin{align*} \amp \amp \Bigl(A(BC)\Bigr)_{ij}=\sum_{\ell=1}^ra_{i\ell}(BC)_{\ell j} =\sum_{\ell=1}^ra_{i\ell}\left(\sum_{k=1}^sb_{\ell k}c_{kj}\right)\\ \amp =\amp \sum_{\ell=1}^r\sum_{k=1}^sa_{i\ell}(b_{\ell k}c_{kj}) \ \text{ (dist. prop.) }\\ \amp =\amp \sum_{k=1}^s\sum_{\ell=1}^r(a_{i\ell}b_{\ell k})c_{kj} \ \text{ (comm.+assoc. prop.) }\\ \amp =\amp \sum_{k=1}^s\left(\sum_{\ell=1}^ra_{i\ell}b_{\ell k}\right)c_{kj} \ \text{ (dist. prop.) }\\ \amp =\amp \sum_{k=1}^s(AB)_{ik}c_{kj} =\Bigl((AB)C\Bigr)_{ij} \end{align*}

This proves their \(ij\)-th entries are equal, and hence \(A(BC)=(AB)C\text{.}\)

Zero matrix and additive inverses.

Another similarity between matrix algebra and real number algebra is the existence of additive inverses.

Definition:.

the \(m\times n\) zero matrix is the matrix \(\boldzero_{m\times n}\) all of whose entries are 0.

Note we have a different zero matrix for every possible size of matrix.

Theorem 1.5.2.

Let \(A=[a_{ij}]\) be \(m\times n\text{.}\) Then the matrix \(-A:=(-1)A=[-a_{ij}]\) is an additive inverse of \(A\) in the sense that

\begin{equation*} A+(-A)=(-A)+A=\boldzero_{m\times n}\text{.} \end{equation*}
Comment:.

thus every \(m\times n\) matrix has an additive inverse. This allows us to “cancel” matrices additively. Example: solve the following equation for \(B\text{.}\)

\begin{align*} A+B\amp =\amp 3A\\ (-A)+A+B\amp =\amp (-A)+3A\\ \boldzero_{m\times n}+B\amp =\amp (-1)A+3A\\ B\amp =\amp 2A \end{align*}

I give all the gory details, but now the point should be clear; we can cancel a multiple of \(A\) on both sides, just as we do in normal algebra!

Abnormalities.

There are two very important properties that matrix algebra fails to enjoy.

Theorem 1.5.3.

Matrix multiplication is NOT \alert{commutative}, and there are nontrivial zero divisors. That is:

  1. Fix \(n\text{.}\) There are (many) \(n\times n\) matrices \(A, B\) such that

    \begin{equation*} AB\alert{\ne}BA\text{.} \end{equation*}

    \alert{Example:} verify that \(\begin{bmatrix}1\amp 0\\ 0\amp 0 \end{bmatrix} \begin{bmatrix}0\amp 1\\ 0\amp 0 \end{bmatrix} \ne \begin{bmatrix}0\amp 1\\ 0\amp 0 \end{bmatrix} \begin{bmatrix}1\amp 0\\ 0\amp 0 \end{bmatrix} \text{.}\)

  2. Given matrices \(A_{m\times n}, B_{n\times r}\text{,}\)

    \begin{equation*} AB=0_{m\times r}\alert{\not\Rightarrow} A=0_{m\times n} \text{ or } B=0_{n\times r}\text{.} \end{equation*}

    In English: if the product of two matrices is the zero matrix, we CANNOT conclude that one of them is the zero matrix. \alert{Example:} verify that \(\begin{bmatrix}0\amp 1\\ 0\amp 0 \end{bmatrix} \begin{bmatrix}1\amp 0\\ 0\amp 0 \end{bmatrix} =\begin{bmatrix}0\amp 0\\ 0\amp 0 \end{bmatrix} \text{.}\)

It is worth pointing out how important the second failed property is to matrix algebra.

In normal real number algebra, if \(a\ne 0\) and \(ab=ac\text{,}\) we may “cancel” the \(a\) and conclude \(b=c\text{.}\) Not so in matrix algebra!

Theorem 1.5.4. Failure of cancellation.
\begin{align*} \text{Suppose \(A\ne 0_{m\times n}\), then } AB=AC \amp \alert{\not\Rightarrow}\amp B=C\\ \text{Suppose \(D\ne 0_{s\times t}\), then } BD=CD \amp \alert{\not\Rightarrow}\amp B=D \end{align*}

Let that sink in. Cancellation is one of the most used properties in normal algebra. It is in general no longer available to us when doing matrix algebra.

Examples: (you should verify).

\begin{equation*} \begin{bmatrix}1\amp 1\amp 0\\ 0\amp 0\amp 0 \end{bmatrix} \begin{bmatrix}1\amp 0\\ 0\amp 0\\ 0\amp 0 \end{bmatrix} = \begin{bmatrix}1\amp 1\amp 0\\ 0\amp 0\amp 0 \end{bmatrix} \begin{bmatrix}0\amp 0\\ 1\amp 0\\ 0\amp 0 \end{bmatrix} \end{equation*}
\begin{equation*} \begin{bmatrix}1\amp 0\amp 0\\ 0\amp 0\amp 0 \end{bmatrix} \begin{bmatrix}1\amp 0\\ 1\amp 0\\ 0\amp 0 \end{bmatrix} = \begin{bmatrix}0\amp 1\amp 0\\ 0\amp 0\amp 0 \end{bmatrix} \begin{bmatrix}1\amp 0\\ 1\amp 0\\ 0\amp 0 \end{bmatrix} \end{equation*}
Invertibility.

What allows cancellation of a nonzero \(a\) in the reals is the existence of its multiplicative inverse, the element \(a^{-1}=\frac{1}{a}\text{,}\) which satisfies \(a^{-1}a=1\text{.}\)

Not all nonzero matrices \(A\) have a multiplicative inverse. The ones that do are called invertible. In order to define this properly, we need to identify a matrix that acts as the number one does: i.e., a multiplicative identity.

Definition 1.5.5.

The identity matrix of size \(n\times n\) is the \(n\times n\) matrix \(I_n\) with 1's along the diagonal (i.e., \(a_{ii}=1\)) and 0's everywhere else (i.e., \(a_{ij}=0\) for \(i\ne j\)). For example:

\begin{equation*} I_1=\begin{bmatrix}1 \end{bmatrix} , I_2=\begin{bmatrix}1\amp 0\\ 0\amp 1 \end{bmatrix} , I_3=\begin{bmatrix}1\amp 0\amp 0\\ 0\amp 1\amp 0\\ 0\amp 0\amp 1 \end{bmatrix} , \text{ etc. } \end{equation*}
Theorem 1.5.6. Multiplicative identity.

For any \(A_{m\times n}\) we have

\begin{align*} I_mA\amp =\amp A\\ AI_n\amp =\amp A\text{.} \end{align*}
Proof.

Use either column or row method to perform the multiplications involved!

Invertibility.
Definition 1.5.7.

A square matrix \(A_{n\times n}\) is invertible if there is a matrix \(B_{n\times n}\) such that

\begin{align*} AB\amp =\amp I_n, \text{ and }\\ BA\amp =\amp I_n \end{align*}

In this case we call \(B\) an inverse of \(A\text{,}\) and we say that \(A\) and \(B\) are inverses of one another.

Theorem (uniqueness of inverses):.

an invertible matrix \(A\) has exactly one inverse. That is, if

\begin{equation*} AB=BA=I_n \end{equation*}

and

\begin{equation*} AC=CA=I_n\text{,} \end{equation*}

then \(B=C\text{.}\)

We denote by \(A^{-1}\) the inverse of \(A\text{.}\)

Proof.

Suppose we have such matrices \(B\) and \(C\text{.}\)

Then \(I_n=AB=AC \Rightarrow BAB=BAC \Rightarrow I_nB=I_nC \Rightarrow B=C\text{.}\)

$2\times 2$ matrices.

When is a \(2\times 2\) matrix \(A=\abcdmatrix{a}{b}{c}{d}\) invertible?

If \(ad-cb\ne 0\text{,}\) then we can easily show that

\begin{equation*} A^{-1}=\frac{1}{ad-bc}\abcdmatrix{d}{-b}{-c}{a} \end{equation*}

is the inverse.

Thus

\begin{equation*} ad-bc\ne 0\Rightarrow A \text{ invertible }\text{.} \end{equation*}

It turns out that the converse is also true, that is

\begin{equation*} ad-bc\ne 0\Leftrightarrow A \text{ invertible }\text{.} \end{equation*}

We will prove this a littler further down the line.

Theorem 1.5.8.

Let \(A,B\) be \(n\times n\) matrices.

\begin{equation*} A \text{ and } B \text{ invertible } \Rightarrow AB \text{ invertible }\text{,} \end{equation*}

and in fact \((AB)^{-1}=B^{-1}A^{-1}\text{.}\)

Proof.

In the absence of more theory, we can only prove invertibility by providing an inverse. The last statement offers up a candidate: viz., \(C=B^{-1}A^{-1}\text{.}\) We need only show that \(C\) satisfies the relevant properties:

\begin{align*} C(AB)\amp =\amp (B^{-1}A^{-1})AB=B^{-1}A^{-1}AB=B^{-1}IB=B^{-1}B=I\\ (AB)C\amp =\amp (AB)B^{-1}A^{-1}=ABB^{-1}A^{-1}=AIA^{-1}=AA^{-1}=I\text{.} \end{align*}
Corollary 1.5.9.

More generally,

\begin{equation*} A_1,A_2,\dots A_r \text{ invertible } \Rightarrow (A_1A_2\cdots A_r)\text{ invertible }\text{,} \end{equation*}

and in fact \((A_1A_2\cdots A_r)^{-1}=A_r^{-1}A_{r-1}^{-1}\cdots A_1^{-1}\text{.}\)

Question: do the implication arrows above go the other way?

Answer: we shall see.

Powers of matrices.
Definition 1.5.10.

Let \(A\) be any \(n\times n\) matrix. For any integer \(r\geq 0\) we define the \(r\)-th power of \(A\text{,}\) \(A^{r}\text{,}\) as follows:

  1. \(A^0=I_n\text{.}\)

  2. \(\ds A^r=\underbrace{A\cdot A\cdots A}_{r \text{ times } }\) for \(r>0\text{.}\)

Furthermore, if \(A\) is invertible, we define

\begin{equation*} A^{-r}=\left(A^{-1}\right)^r\text{,} \end{equation*}

for any positive integer \(r>0\text{.}\)

Theorem 1.5.11. Power rules.

  1. \(\displaystyle A^{r}A^s=A^{r+s}\)

  2. \(\displaystyle (A^r)^s=A^{rs}\)

  3. \(\displaystyle \left(A^{-1}\right)^{-1}=A\)

  4. \(\displaystyle \left(A^n\right)^{-1}=A^{-n}\)

  5. \((kA)^n=k^nA^n\) for any scalar \(k\text{.}\)

Question:.

Transposes and inverses.
Theorem 1.5.12.

  1. \(\displaystyle \left(A^T\right)^T=A\)

  2. \(\displaystyle (A+B)^T=A^T+B^T\)

  3. \(\displaystyle (A-B)^T=A^T-B^T\)

  4. \(\displaystyle (kA)^T=kA^T\)

  5. \(\displaystyle (AB)^T=B^TA^T\)

Theorem 1.5.13.

Let \(A_{n\times n}\) be invertible. Then \(A^T\) is invertible, and \(\left(A^T\right)^{-1}=\left(A^{-1}\right)^T\text{.}\)

More sample proofs.

Let's prove some of the last two theorems.

Proof.
\begin{align*} \left((A+B)^T\right)_{ij}\amp =\amp (A+B)_{ji} \ \text{ (def. of transp.) }\\ \amp =\amp A_{ji}+B_{ji} \ \text{ (def. of +) }\\ \amp =\amp (A^T)_{ij}+(B^T)_{ij} \ \text{ (def. of transp.) }\\ \amp =\amp (A^T+B^T)_{ij} \ \text{ (def. of +) } \end{align*}

Since \(ij\)-entry of LHS and RHS is same for each \((i,j)\text{,}\) it follows that \((A+B)^T=A^T+B^T\text{.}\)

Proof.
