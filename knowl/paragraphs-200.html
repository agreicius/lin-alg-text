Dimension theorem compendium.

We collect here some of the most important results from this section.

Theorem 3.6.5.

{Dimension theorem compendium} Let \(V\) be a vector space, and suppose \(B=\{\boldv_1,\boldv_2,\dots, \boldv_n\}\) is a basis for \(V\text{.}\)

  1. (Street smarts) If \(S\) is a subset of \(V\) containing less than \(n\) elements, then \(S\) does not span \(V\text{.}\)

  2. (Street smarts) If \(S\) is a subset of \(V\) containing more than \(n\) elements, then \(S\) is linearly dependent.

  3. All bases of \(V\) contain exactly \(n\) elements.

  4. If \(S\) is linearly independent, then \(S\) can be extended to a basis of \(V\text{.}\)

  5. If \(S\) spans \(V\text{,}\) then some subset of \(S\) is a basis for \(V\text{:}\) i.e., \(S\) can be contracted to a basis.

  6. If \(\val{S}=n\text{,}\) then \(S\) is a basis if and only if \(S\) is linearly independent if and only if \(S\) spans \(V\text{.}\)

  7. (Dimension of subspaces I) Given subspace \(W\subseteq V\text{,}\) we have \(\dim W\leq \dim V=n\text{.}\)

  8. (Dimension of subspaces II) Given subspace \(W\subseteq V\text{,}\) we have

    \begin{equation*} W=V \text{ if and only if } \dim W=\dim V\text{.} \end{equation*}
