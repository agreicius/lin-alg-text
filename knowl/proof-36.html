Proof

First observe that \(\boldzero\) is the zero function: the function that assigns 0 to all inputs \(x\text{.}\)

We ask whether there is a nontrivial combination \(af+bg+ch=\boldzero\text{.}\)

Key observation: the equality above is an equality of functions. Thus this is true if and only if \(af(x)+bg(x)+ch(x)=0\) \alert{for all \(x\)}.

To get a linear system, we evaluate the above at a few clever choices of \(x\text{:}\)

\begin{align*} x=0\amp :\amp a(0)+b\cos(0)+c\sin(0)=0\Rightarrow 0+b+0=0\Rightarrow b=0\\ x=\pi\amp :\amp a(\pi)+c\sin(\pi)=0\Rightarrow \pi a+0=0\Rightarrow a=0 \end{align*}

Having shown \(a=b=0\text{,}\) we are left with the equation \(c\sin(x)=0\) for all \(x\text{,}\) which is true iff \(c=0\text{.}\)

Thus the only linear combination yielding \(\boldzero\) is \(a=b=c=0\text{,}\) the trivial one, showing \(S\) is linearly independent.
