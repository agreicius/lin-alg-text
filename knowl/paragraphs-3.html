Linear equations.
Definition 1.1.1.

A linear equation in the \(n\) variables (or unknowns) \(x_1,x_2,\dots x_n\) is an equation that can be written in the form

\begin{equation*} a_1x_1+a_2x_2+\cdots a_nx_n=b\text{,} \end{equation*}

where \(a_1,\dots, a_n, b\) are constants.

The linear equation is called homogenous when \(b=0\text{.}\)

Examples.

  1. Consider \(\sqrt{3}x+\sin(5)=2z-e^4y\text{.}\) This is a linear equation in the unknowns \(x,y,z\) as we can write it as \(\sqrt{3}x+e^4y-2z=-\sin(5)\text{.}\) Note that this is a nonhomogenous linear equation.

  2. The equation \(x^2+y^2=1\) is a nonlinear equation in the unknowns \(x\) and \(y\text{.}\)
