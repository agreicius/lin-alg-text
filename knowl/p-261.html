In particular, a homogeneous linear system

\begin{equation*} \homsys \end{equation*}

can be represented as the single matrix equation

\begin{equation*} A\boldx=\underset{m\times 1}{\boldzero}\text{,} \end{equation*}

where \(A=[a_{ij}]\) and \(\boldx=[x_i]\text{;}\) and \((s_1,s_2,\dots, s_n)\) is a solution to the homogenous system if and only if its corresponding column vector \(\underset{n\times 1}{\boldsymbol}=[s_i]\) is a solution to the matrix equation \(A\boldx=\underset{m\times 1}{\boldzero}\text{.}\)
