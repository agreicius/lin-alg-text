Solutions to linear systems.
Definition 1.1.3.

A solution to a linear equation

\begin{equation*} a_1x_1+a_2x_2+\cdots a_nx_n=b \end{equation*}

is a sequence (or ordered \(n\)-tuple) \((s_1,s_2,\dots, s_n)\) for which the substitution \(x_i=s_i\) makes the equation true. We say \((s_1,\dots ,s_n)\) solves the equation in this case .

A solution to a system of linear equations

\begin{equation*} \eqsys \end{equation*}

is a sequence (or ordered \(n\)-tuple) \((s_1,s_2,\dots, s_n)\) which solves all \(m\) of the system's equations. We say \((s_1,s_2,\dots, s_n)\) solves the system in this case.
