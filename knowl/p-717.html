Let \(V\) be finite dimensional.

  1. If \(S\) spans \(V\text{,}\) then some subset of \(S\) is a basis of \(V\text{.}\) (Basically, throw out the redundant members!).

  2. If \(S\) is linearly independent, then \(S\) can be extended to a basis of \(V\text{.}\) (Basically, add vectors to \(S\) one by one until it spans.)

  3. Suppose \(\dim V=n\text{,}\) and let \(S=\{\boldv_1,\dots ,\boldv_n\}\) be any set of \(n\) vectors in \(V\text{.}\) Then

    \begin{equation*} S \text{ spans \(V\) } \Leftrightarrow S \text{ is linearly independent }\text{.} \end{equation*}

    Thus if we know beforehand that \(\dim V=n\text{,}\) then to show a given set \(S=\{\boldv_1,\dots ,\boldv_n\}\) is a basis it suffices to show either that is spans, or that it is linearly independent.
